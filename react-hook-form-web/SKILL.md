# React Hook Form Web

Use this skill as a basic starting point.

## Workflow

1. Review the current React form structure and validation needs.
2. Identify field registration, error handling, and submission flow.
3. Use React Hook Form patterns that fit the existing components.
4. Keep validation and field state explicit.
5. Verify form behavior and error states.

## Output

- Produce practical React Hook Form changes.
- Keep forms predictable and maintainable.
- Prefer explicit validation and state handling.
