# Docusaurus Web

Use this skill as a basic starting point.

## Workflow

1. Review the docs structure, navigation, and theme configuration.
2. Identify where content, sidebars, or plugins need changes.
3. Keep docs organization easy to scan and maintain.
4. Preserve consistent formatting and navigation behavior.
5. Verify local docs rendering and link integrity.

## Output

- Produce practical documentation site changes.
- Keep docs maintainable and developer friendly.
- Prefer clear information architecture over scattered pages.
