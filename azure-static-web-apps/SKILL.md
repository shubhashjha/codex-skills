# Azure Static Web Apps

Use this skill as a basic starting point.

## Workflow

1. Review the frontend build output and deployment needs.
2. Identify app location, output location, and routing behavior.
3. Keep configuration aligned with static hosting constraints.
4. Match deployment steps to the actual repository structure.
5. Verify hosted routes, assets, and environment behavior.

## Output

- Produce practical Azure static deployment changes.
- Keep hosting configuration simple and maintainable.
- Prefer clear build and deploy paths over implicit setup.
