# Husky Lint Staged Web

Use this skill as a basic starting point.

## Workflow

1. Review the current package scripts and linting setup.
2. Identify which checks belong in pre-commit hooks.
3. Keep hook execution fast and scoped to staged files.
4. Avoid adding fragile or slow checks to local hooks.
5. Verify developer workflow after configuration changes.

## Output

- Produce practical pre-commit automation changes.
- Keep hooks lightweight and reliable.
- Prefer fast feedback over excessive local enforcement.
