# Skeleton Loading Web

Use this skill as a basic starting point.

## Workflow

1. Review current loading states and user waiting points.
2. Identify where skeletons improve perceived responsiveness.
3. Match placeholders to real content structure.
4. Keep loading transitions subtle and clear.
5. Verify layout stability during data loading.

## Output

- Produce practical skeleton loading improvements.
- Keep placeholders consistent and lightweight.
- Prefer realistic structure over decorative placeholders.
