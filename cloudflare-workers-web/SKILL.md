# Cloudflare Workers Web

Use this skill as a basic starting point.

## Workflow

1. Review the runtime constraints and request handling needs.
2. Identify edge logic, routing, and environment boundaries.
3. Keep handlers small and explicit.
4. Respect worker platform limitations and deployment patterns.
5. Verify runtime behavior locally and after deployment.

## Output

- Produce practical edge runtime changes.
- Keep worker logic lightweight and maintainable.
- Prefer explicit request handling over hidden abstractions.
