# Infinite Scroll Web

Use this skill as a basic starting point.

## Workflow

1. Review the current data fetching and list rendering logic.
2. Identify pagination rules, loading states, and end-of-list behavior.
3. Use scroll observers or pagination patterns that fit the app.
4. Keep loading feedback and error states clear.
5. Verify behavior on long lists and mobile devices.

## Output

- Produce practical infinite scroll changes.
- Keep the experience performant and understandable.
- Prefer robust pagination handling over fragile scroll logic.
