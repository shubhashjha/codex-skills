# Modal Dialogs Web

Use this skill as a basic starting point.

## Workflow

1. Review the current interaction and state management model.
2. Identify whether modal behavior is the right UX choice.
3. Keep focus management and close behavior accessible.
4. Match dialog layout to the existing design system.
5. Verify keyboard, backdrop, and mobile behavior.

## Output

- Produce practical modal dialog changes.
- Keep dialogs accessible and easy to maintain.
- Prefer simple flows over nested modal complexity.
