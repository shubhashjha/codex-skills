# Master Data Management

Use this skill as a basic starting point.

## Workflow

1. Review the current shared reference entities and ownership boundaries.
2. Identify duplication, synchronization, or consistency risks.
3. Keep authoritative data ownership explicit.
4. Avoid spreading write access across unrelated workflows.
5. Verify consistency across representative data consumers.

## Output

- Produce practical master data improvements.
- Keep ownership and consistency rules clear.
- Prefer explicit authoritative sources over duplicated data entry.
