# Helmet Security Web

Use this skill as a basic starting point.

## Workflow

1. Review the current Express or Node security middleware setup.
2. Identify missing protections and compatibility constraints.
3. Apply Helmet configuration that fits the app requirements.
4. Avoid breaking legitimate embeds or scripts unintentionally.
5. Verify headers in local responses after changes.

## Output

- Produce practical HTTP security improvements.
- Keep configuration explicit and reviewable.
- Prefer safe defaults with intentional exceptions.
