# Image Optimization Web

Use this skill as a basic starting point.

## Workflow

1. Review current image usage, sizes, and loading behavior.
2. Identify oversized, blocking, or poorly formatted assets.
3. Use responsive images, compression, and lazy loading as appropriate.
4. Preserve visual quality where it matters.
5. Verify improvements across devices and screen sizes.

## Output

- Produce practical image performance improvements.
- Keep asset delivery efficient and maintainable.
- Prefer balanced quality and speed.
