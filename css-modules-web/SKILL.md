# CSS Modules Web

Use this skill as a basic starting point.

## Workflow

1. Review the current component and styling structure.
2. Identify reusable class names and scope boundaries.
3. Apply CSS Modules in a way that fits existing conventions.
4. Keep selectors simple and predictable.
5. Verify styles across interactive and responsive states.

## Output

- Produce practical scoped styling changes.
- Keep styles modular and easy to trace.
- Prefer clarity over deeply nested selectors.
