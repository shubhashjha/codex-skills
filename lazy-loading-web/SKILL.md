# Lazy Loading Web

Use this skill as a basic starting point.

## Workflow

1. Review the current loading strategy and bundle structure.
2. Identify safe candidates for lazy loading.
3. Apply lazy loading without breaking navigation or UX.
4. Provide fallback states where needed.
5. Verify performance and loading behavior.

## Output

- Produce practical lazy loading improvements.
- Keep the user experience smooth and predictable.
- Prefer measurable performance wins over premature optimization.
