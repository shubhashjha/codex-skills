# Charts Visualization Web

Use this skill as a basic starting point.

## Workflow

1. Review the data shape and visualization goal.
2. Choose the simplest chart type that communicates clearly.
3. Keep labels, legends, and scales readable.
4. Match the current UI and color system.
5. Verify responsiveness and data edge cases.

## Output

- Produce practical chart and dashboard changes.
- Keep visualizations readable and honest.
- Prefer clarity over visual density.
