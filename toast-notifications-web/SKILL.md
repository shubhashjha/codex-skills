# Toast Notifications Web

Use this skill as a basic starting point.

## Workflow

1. Review the current feedback patterns and error handling.
2. Identify where toast notifications improve user clarity.
3. Keep toast timing, severity, and placement consistent.
4. Avoid overwhelming the user with redundant messages.
5. Verify behavior across success, warning, and error flows.

## Output

- Produce practical notification feedback changes.
- Keep messaging concise and consistent.
- Prefer clear user feedback over noisy UI.
