# Cloudflare Pages Web

Use this skill as a basic starting point.

## Workflow

1. Review the build process and output directory.
2. Identify routing, asset, and environment requirements.
3. Keep deployment setup compatible with static edge hosting.
4. Avoid platform-specific configuration unless justified.
5. Verify deployed asset paths and page routing.

## Output

- Produce practical Cloudflare Pages deployment changes.
- Keep configuration simple and edge friendly.
- Prefer static-first hosting patterns where possible.
