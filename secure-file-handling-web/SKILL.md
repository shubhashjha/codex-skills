# Secure File Handling Web

Use this skill as a basic starting point.

## Workflow

1. Review the file flow, storage destination, and user interaction path.
2. Identify file type, size, naming, and access constraints.
3. Keep validation and storage rules explicit.
4. Avoid unsafe assumptions about file content or extensions.
5. Verify upload, access, and rejection behavior.

## Output

- Produce practical secure file handling changes.
- Keep file workflows safe and maintainable.
- Prefer explicit validation and storage controls.
