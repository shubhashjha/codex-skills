# Chakra UI Web

Use this skill as a basic starting point.

## Workflow

1. Review the Chakra UI component structure and theme usage.
2. Identify the current layout and spacing approach.
3. Apply changes using Chakra primitives and tokens.
4. Keep style overrides limited and intentional.
5. Verify accessibility and responsive behavior.

## Output

- Produce practical Chakra UI based changes.
- Keep styles consistent with the theme system.
- Prefer composable primitives over custom ad hoc markup.
