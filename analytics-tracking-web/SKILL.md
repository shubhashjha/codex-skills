# Analytics Tracking Web

Use this skill as a basic starting point.

## Workflow

1. Review the user journeys and analytics goals.
2. Identify meaningful events, properties, and tracking boundaries.
3. Keep naming consistent and events tied to product actions.
4. Avoid tracking sensitive or low-value data unnecessarily.
5. Verify events fire correctly in realistic flows.

## Output

- Produce practical analytics instrumentation changes.
- Keep tracking intentional and maintainable.
- Prefer meaningful events over excessive telemetry.
