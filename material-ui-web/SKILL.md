# Material UI Web

Use this skill as a basic starting point.

## Workflow

1. Review the existing Material UI setup and component usage.
2. Identify the current theming and layout conventions.
3. Use the smallest useful component or style change.
4. Keep spacing, typography, and interaction patterns consistent.
5. Verify responsive behavior and component states.

## Output

- Produce practical Material UI based changes.
- Keep the result easy to review and extend.
- Prefer consistent theming over one-off overrides.
