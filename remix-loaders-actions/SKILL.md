# Remix Loaders Actions

Use this skill as a basic starting point.

## Workflow

1. Review the current route modules and data flow.
2. Identify loader and action responsibilities clearly.
3. Keep validation, errors, and redirects explicit.
4. Preserve route level separation of concerns.
5. Verify request handling and mutation flows.

## Output

- Produce practical Remix route data changes.
- Keep loaders and actions readable and maintainable.
- Prefer explicit request handling over hidden side effects.
