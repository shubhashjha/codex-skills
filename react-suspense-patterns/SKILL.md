# React Suspense Patterns

Use this skill as a basic starting point.

## Workflow

1. Review the current async data and component loading model.
2. Identify safe suspense boundaries and fallback needs.
3. Keep loading states understandable and localized.
4. Avoid hiding important control flow behind complex patterns.
5. Verify rendering and fallback transitions.

## Output

- Produce practical Suspense related changes.
- Keep async UI boundaries explicit and understandable.
- Prefer clear loading behavior over experimental complexity.
