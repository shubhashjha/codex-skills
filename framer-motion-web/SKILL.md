# Framer Motion Web

Use this skill as a basic starting point.

## Workflow

1. Review the existing React component structure and motion usage.
2. Identify where motion improves UX without harming performance.
3. Use Framer Motion primitives that fit the current component model.
4. Keep transitions consistent and easy to reason about.
5. Verify animation behavior on desktop and mobile.

## Output

- Produce practical Framer Motion based UI changes.
- Keep animation readable and maintainable.
- Prefer small, intentional interactions over large effects.
