# Markdown Rendering Web

Use this skill as a basic starting point.

## Workflow

1. Review the source content format and rendering requirements.
2. Identify security, sanitization, and formatting constraints.
3. Keep markdown output readable and consistent with the UI.
4. Handle code blocks, lists, links, and media intentionally.
5. Verify rendering with realistic content samples.

## Output

- Produce practical markdown rendering changes.
- Keep content safe, readable, and maintainable.
- Prefer clear formatting over excessive customization.
