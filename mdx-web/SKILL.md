# MDX Web

Use this skill as a basic starting point.

## Workflow

1. Review the current content pipeline and component usage.
2. Identify where MDX improves authoring or presentation.
3. Keep embedded components predictable and reusable.
4. Preserve readable content structure.
5. Verify rendered output and content formatting.

## Output

- Produce practical MDX based content changes.
- Keep content and component boundaries clear.
- Prefer maintainable authoring patterns over heavy customization.
