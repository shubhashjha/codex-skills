# codex-skills

A starter repository of common Codex skills for .NET software development work.

This repository contains basic skill folders with plain descriptions and short workflow notes. The content is intentionally simple so it can be used as a starting point and edited later.

## Structure

- Each skill is stored in its own folder.
- Each folder contains a `SKILL.md` file with metadata and a short instruction body.
- The wording is intentionally plain and generic.

## Notes

- This is a starter collection, not a finished framework.
- Adjust descriptions and guidance to match your own team or product standards.
- The folder has already been initialized as a local Git repository.
