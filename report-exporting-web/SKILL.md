# Report Exporting Web

Use this skill as a basic starting point.

## Workflow

1. Review the report source data and export format requirements.
2. Identify filters, permissions, and export size constraints.
3. Keep export generation predictable and user focused.
4. Avoid blocking the UI with heavy export work where unnecessary.
5. Verify output correctness with representative data.

## Output

- Produce practical export related changes.
- Keep report downloads reliable and maintainable.
- Prefer explicit export flows over hidden background behavior.
