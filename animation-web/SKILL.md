# Animation Web

Use this skill as a basic starting point.

## Workflow

1. Review the existing UI structure, CSS, and JavaScript patterns.
2. Identify where animation improves clarity, feedback, or perceived quality.
3. Prefer subtle, performant motion over decorative effects.
4. Keep timing, easing, and interaction states consistent.
5. Verify responsiveness, accessibility, and reduced-motion behavior.

## Output

- Produce practical animation focused changes.
- Keep motion purposeful and performant.
- Prefer simple, maintainable patterns over complex effects.
