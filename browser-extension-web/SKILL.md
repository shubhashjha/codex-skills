# Browser Extension Web

Use this skill as a basic starting point.

## Workflow

1. Review the extension architecture, manifest version, and target browser needs.
2. Identify responsibilities across popup, background, and content scripts.
3. Keep permissions minimal and explicit.
4. Match the extension UI and behavior to the intended workflow.
5. Verify extension behavior in the browser environment.

## Output

- Produce practical browser extension changes.
- Keep extension boundaries secure and maintainable.
- Prefer minimal permissions and clear structure.
