# TanStack Table Web

Use this skill as a basic starting point.

## Workflow

1. Review the current data model and table requirements.
2. Identify sorting, filtering, pagination, or selection needs.
3. Keep column definitions and state handling explicit.
4. Match the current styling and interaction patterns.
5. Verify behavior with realistic data sizes.

## Output

- Produce practical table focused UI changes.
- Keep configuration readable and maintainable.
- Prefer explicit behavior over hidden complexity.
