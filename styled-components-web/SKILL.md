# Styled Components Web

Use this skill as a basic starting point.

## Workflow

1. Review the current React and styling patterns.
2. Identify where styled-components improves maintainability.
3. Keep component styles small and colocated.
4. Reuse theme tokens and shared values where possible.
5. Verify style behavior across states and layouts.

## Output

- Produce practical styled-components based changes.
- Keep styles readable and component focused.
- Prefer reuse over duplicated style blocks.
