# Contentlayer Web

Use this skill as a basic starting point.

## Workflow

1. Review the content sources and schema requirements.
2. Identify document types, fields, and generated data needs.
3. Keep content models explicit and easy to evolve.
4. Match generated content usage to the current app patterns.
5. Verify build output and content queries.

## Output

- Produce practical content modeling changes.
- Keep schemas readable and maintainable.
- Prefer explicit content structure over implicit conventions.
