# Stripe Payments Web

Use this skill as a basic starting point.

## Workflow

1. Review the payment flow and backend responsibilities.
2. Identify checkout, webhook, and status handling needs.
3. Keep sensitive operations on the server side where required.
4. Match the current UX and pricing model.
5. Verify success, failure, and cancellation flows.

## Output

- Produce practical payment integration changes.
- Keep payment handling secure and maintainable.
- Prefer explicit payment states over hidden assumptions.
